# Avatar Images

Add your avatar images here. The system expects 12 avatar images named:

- `avatar-1.png`
- `avatar-2.png`
- `avatar-3.png`
- `avatar-4.png`
- `avatar-5.png`
- `avatar-6.png`
- `avatar-7.png`
- `avatar-8.png`
- `avatar-9.png`
- `avatar-10.png`
- `avatar-11.png`
- `avatar-12.png`

## Image Requirements

- Format: PNG (recommended) or JPG
- Size: 200x200px or larger (square aspect ratio)
- File size: Keep under 100KB for optimal performance

## Adding Avatars

1. Add your avatar images to this directory with the naming convention above
2. The avatar selector will automatically display them
3. If an image is missing, a numbered placeholder will be shown
